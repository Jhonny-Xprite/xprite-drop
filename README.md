# Xprite-Drop

Full-stack SaaS application built with **Synkra AIOX** — AI-Orchestrated System for Full Stack Development.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ (v24.14.0 ✅)
- GitHub CLI (v2.87.3 ✅)
- Git (v2.53.0 ✅)
- Supabase CLI (v2.75.0 ✅)
- Railway CLI (v4.30.5 ✅)
- Docker (v29.2.1 ✅)
- pnpm (installed ✅)
- bun (installed ✅)

### Setup

```bash
# Clone repository
git clone https://github.com/Jhonny-Xprite/xprite-drop.git
cd xprite-drop

# Install dependencies
npm install
# or
pnpm install

# Development
npm run dev
```
